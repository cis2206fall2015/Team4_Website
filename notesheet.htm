Files

ls –al                                                      Shows All information about directories and its files

pwd                                                       Shows the path to current directory

mkdir directory-name                    Create a directory

rm file-name                                      Delete a file

rm –rf directory-name                   Delete a directory

rm –f file-name                                force delete a files

cp file1 file2                                        Copy files

mv file1 file2                                      Move file

cat file                                                   output file

chmod                                                  change owner of file

cd ..                                                        Goes up one directory level

cd                                                           go to home directory

cd /test                                                change to /test directory

 

Archives

tar cf home.tar home                     Creates tar call home containing home

tar xf file.tar                                       Extract tar

 

Login

ssh –p port user@host  connect User to host on port

 

Disk Usage

df –h                     Show free space

df –sh                   Show free space in current directory

 

System Commands

uptime                                 Shows How Long the System Has been Running

hostname                           Shows System Host Name

hostname –i                      Shows Host Ip

last reboot                          Shows Last Reboot

date                                      Shows Date and Time

whoami                               your current log in

 

User Commands

who                                       show who is logged on

 

 

Networking

ifconfig –a                           Display all network ports and ip’s

ifconfig eth0                       Display Ethernet port

ping host                             Pings host to test connection

whois domain                    retrieves who is info from domain

wget file                              Download file
