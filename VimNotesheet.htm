Vim Commands

Movement Commands

Command and Action

h, j, k, l

left, up down, right

), (

Beginning of next, previous sentence

RETURN or ENTER

First nonblank character of next line

+, -

First nonblank character of next, previous line

M

Middle line of screen

nL

n (number) of lines before last line

CTRL-E, CTRL-Y

Show one more line at bottom, top of window

z -

Reposition line with cursor: to bottom of screen

?pattern

Search backward for a match

fx

Search forward for character x in current line

Tx

Search backward for character after x in current line

CTRL - G

Display current line number

:n

Move to line n in file

` `

Return to previous mark of context

Command and Action

w, W b, B

Forward, backward by word

}, {

Beginning of next, previous paragraph

0, $

First, last position of next line

n |

Column n of current line

L

Last line of screen

CTRL-F, CTRL B

Scroll forward, backward one screen

z RETURN

Reposition line with cursor: to top of screen

CTRL - L

Redraw screen (without scrolling)

n, N

Repeat last search in name, opposite direction

Fx

Search backward for character x in current line

;

Repeat previous current-line search

nG

Move to line number n

mx

Mark current position as x

x

Move to beginning of line containing mark x

Command and Action

e, E

End of Word

]], [[

Beginning of next, previous section

^

First nonblank character of current line

H

Top line of screen

nH

n (number) of lines after top line

CTRL-D, CTRL-U

Scroll down, up one-half screen

z .

Reposition line with cursor: to middle of screen

/pattern

Search forward for a match

/, ?

Repeat last search forward, backward

tx

Search forward for character before x in current line

Repeat previous current-line search in opposite direction

G

Move to last line in file

`x

Move cursor to mark x

Return to beginning of line containing previous mark

EDITING Commands

Command and Action

i, a

Insert text before, after cursor

r

Replace character

cmotion

Change text between the cursor and the target of motion

s

Substitute: delete character and insert new text

X

Delete character before cursor

dmotion

Delete text between the cursor and the target of motion

np

Put text from delete buffer number n after cursor

ayy

Yank current line into named buffera (a-z). Uppercase names append text

aP

Put text from buffer a before (a-z)

J

Join two lines

:co OR :t

Copy lines

Command and Action

I, A

Insert text before beginning, after end of line

cw

Change word

C

Change to end of line

S

Substitute: delete current line and insert new text

dw

Delete word

D

Delete to end of line

yw

Yank (copy) word

ymotion

Yank text between the cursor and the target of motion

.

Repeat last edit command

:d

Delete lines

:.,$d

Delete from current line to end of file

:.,/pattern/co$

Copy from current line through line containing pattern to end of file

Command and Action

o, O

Open new line for text below, above cursor

cc

Change current line

R

Type over (OVERWRITE) characters

x

Delete character under cursor

dd

Delete current line

p, P

Put deleted text after, before cursor

yy

Yank current line

p, P

Put yanked text after, before cursor

u, U

Undo last edit; restore current line

:m

Move Lines

:30,60m0

Move lines 30 through 60 to top of file

 

Exit Commands

Command and Action

ZZ or :x or :wq

Save and quit

:30, 60w newfile

Save lines 30 to 60 as newfile

:q!

Quit, overriding protection

Command and Action

:w

Save

:30, 60w>>file

save from line 30 to 60 and append to file

Q

Quit vi and invoke ex

Command and Action

:w!

Save, overriding protection

:q

Quit

:e!

Return to version of current file at time of last save
